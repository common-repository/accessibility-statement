== Description ==
Accessibility statement: Latvian
Generate an accessibility statement for your website in compliance with Latvian legislation.
This plugin helps you create an accessibility statement, a separate page, for your Latvian website. You can freely customize its content. The page is in Latvian and adjusted to reflect requirements defined by the legal acts of the Republic of Latvia.
This plugin is using Advanced Custom Fields, though ACF is not a prerequisite to use the plugin.

== Installation ==
1. **Visit** Plugins > Add New
2. **Search** for \"Accessibility statement\"
3. **Install and Activate** Accessibility statement from your Plugins page
4. **Click** on the  menu item Pages and press Edit on page with title \"Piekļūstamības paziņojums\"
5.  Change dummy field values to real content and **Click** on \"Update post\" to save changes and generate content
6.  Generated content can be found at http://example.com/pieklustamibas-pazinojums/
6. **Click** on the menu item Settings >  \"Accessibility statement settings\" for advanced content editing.
